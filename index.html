Tentang Saya

Saya adalah mahasiswa aktif Program Studi Sistem Informasi di Universitas Terbuka yang memiliki ketertarikan kuat pada bidang teknologi informasi, khususnya dalam analisis sistem, pemrograman, dan pengembangan aplikasi. Dengan latar belakang pendidikan yang menekankan kemandirian dan kedisiplinan, saya terbiasa mengelola waktu secara efektif serta menyelesaikan tugas dan proyek secara terstruktur.
Saya memiliki pengalaman dalam mengerjakan proyek berbasis web, dokumentasi sistem, serta penggunaan berbagai bahasa pemrograman dan tools pendukung pengembangan perangkat lunak. Selain kemampuan teknis, saya juga mengutamakan komunikasi yang baik, kolaborasi tim, dan semangat belajar yang berkelanjutan. Ke depannya, saya berkomitmen untuk terus mengembangkan keahlian di bidang teknologi informasi dan memberikan kontribusi nyata dalam mendukung transformasi digital di berbagai sektor.
Saya percaya bahwa teknologi bukan hanya tentang kode, tetapi tentang menciptakan perubahan. Karena itu, saya terus mengasah keterampilan saya dengan tujuan menjadi seorang web developer dan software engineer yang profesional. Bagi saya, setiap baris kode adalah langkah kecil menuju masa depan yang lebih cerdas dan terhubung.

Proyek Saya

Proyek 1

Proyek: Website Kedai Kopi


Saya merancang dan membangun sebuah website sederhana untuk kedai kopi lokal menggunakan HTML dan CSS. Website ini mencakup halaman beranda, menu produk, profil kedai, dan formulir kontak. Fokus utama proyek ini adalah memberikan tampilan yang menarik, user-friendly, dan informatif agar pelanggan dapat dengan mudah mengenal produk serta layanan kedai kopi tersebut. Proyek ini juga menjadi latihan penting dalam menerapkan prinsip dasar desain web dan struktur halaman yang responsif.

Proyek 2

Proyek: Analisis Data Penjualan Sederhana


Dalam proyek ini, saya menganalisis data penjualan menggunakan Microsoft Excel dan Google Sheets untuk menemukan pola penjualan tertinggi berdasarkan produk dan waktu. Saya menggunakan teknik dasar seperti filter data, pivot table, dan pembuatan grafik visual untuk menyajikan informasi dengan lebih jelas. Proyek ini melatih saya dalam membaca data, menarik kesimpulan, dan menyusun laporan yang bisa dijadikan dasar pengambilan keputusan bisnis.
